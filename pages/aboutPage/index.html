Project Overview

This project consisted of a website that contained three pages. The purpose of these pages is to work with APIs, Javascript, and Bootstrap.

Outcome

The Album Search page gives users the ability to search artists from the iTunes store. Upon a search, users will be shown a list of the top 5 albums from the input artist. Itunes' API was worked with to get a JSON response with results

On the About page, a weather API was used. The user's location is used to get a JSON response containing that location's current weather conditions.
